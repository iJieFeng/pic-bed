# pic-bed

### github 访问
可使用 Staticaly CDN 进行加速，规则如下：

```
https://cdn.staticaly.com/gh/user/repo@version/file
```
示例

```
https://cdn.staticaly.com/gh/iJieFeng/pic-bed@main/img/404.jpg
```

user是用户名  repo是仓库名  version代表版本(tag或者分支 默认为main)  flie是文件路径
